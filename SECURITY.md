# Security Policy

## Scope

This project is a static site (Astro SSG) deployed on Cloudflare Pages.
There is no user authentication, database, or server-side processing, so the attack surface is minimal.

## Supported Versions

Only the latest version deployed from the `main` branch is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through [GitHub Security Advisories](https://github.com/s-hirano-ist/s-public/security/advisories/new).

Please do not open a public issue for security vulnerabilities.
